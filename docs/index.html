Документація

Як локалізувати банер
за 5 кліків

Покрокова інструкція для дизайнерів і маркетологів. Готуєш таблицю перекладів — прогоняєш через адаптер — натискаєш Apply у Photoshop. Жодних копі-пастів, жодного ручного підбору розміру тексту.

0Швидкий старт

Якщо все під рукою — 5 хвилин від «маю таблицю» до «маю готову локалізацію в PSD».

  1. 1

    Підготуй таблицю

    Google Sheets / Excel / CSV. Перший стовпчик — ключі (title, cta...), решта — мови (ru, en, ar...). Якщо таблиця хаотична — не страшно, адаптер впорається.

  2. 2

    Прогоняй через адаптер

    Відкривай locality-csv.richasez666.workers.dev, вставляй URL таблиці або файл — натискай Розібрати.

  3. 3

    Завантаж готовий CSV

    Адаптер віддасть чистий файл у форматі плагіна. Просто скачай його.

  4. 4

    Відкрий PSD у Photoshop

    Майстер-документ з шарами, іменованими як ключі (title, cta...). Плагін Locality має бути встановлений (див. розділ нижче).

  5. 5

    Apply → готово

    У панелі Locality обери CSV-файл і мову → Apply. Текст підставиться, кегль автоматично підлаштується якщо переклад довший за оригінал.

Локалізації

1Інструмент: CSV Адаптер

Що це і навіщо

Плагін Locality читає один формат: перший стовпчик key, решта — ISO-коди мов. Реальні маркетингові таблиці виглядають інакше: повні назви країн, гендерні маркери, банер-розміри в окремих колонках, транспоновані шити, абревіатури мов.

Адаптер бере цей хаос на вхід — і віддає чистий CSV у форматі плагіна. Розпізнає:

  • Повні назви мов: Russian, русский, українська, arabic, bengali — і ще 80+ варіантів
  • Описові коди: Argentina (spanish) + Landing → Argentina_Spanish
  • Транспоновані таблиці (мови в рядках, ключі в колонках)
  • Метадані: рядки «оновлені коди», стовпчики Н/Ж, банер-розміри (120x240, 320х100 моб...)
  • Match/Variant-патерни для заміни команд або варіантів креативу
  • Багаторядкові клітинки, нестандартні апострофи, кириличне «х» у розмірах

Як завантажити дані — 3 способи

Спосіб A

Вставити з буфера

Вкладка Вставити CSV. Скопіюй з Google Sheets чи Excel — і встав сюди прямо текст із табуляціями або комами.

Підійде коли: маленька таблиця, нема публічного шиту.

Спосіб B

URL Google Sheets

Вкладка URL. Зроби шит публічним (Share → Anyone with link) — вставляй будь-який лінк: /edit, /pub, /pubhtml. Адаптер сам конвертує у CSV-export.

Підійде коли: таблицю редагує команда, треба завжди останню версію.

Спосіб C

Файл

Вкладка Файл. Дропни .csv, .xlsx, .tsv, .docx, .pdf. Якщо в xlsx кілька аркушів — обери потрібний у пікері.

Підійде коли: таблиця у файлі, нема онлайн-версії.

Що робить «Розібрати»

Натиснення кнопки запускає послідовність розумних чисток:

locality-csv.richasez666.workers.dev

Вставити URL Файл
https://docs.google.com/spreadsheets/d/1oU9fjHzJ...
  1. Викидає шум — колонки з banner-розмірами (120x240, 320х100) автоматично відсіюються.
  2. Знаходить шапку — пропускає «метадані» зверху (рядки типу «оновлені коди» чи декоративні написи), знаходить рядок із ключами.
  3. Чистить ліву колонку — Н/Ж, M/F та інші 1-2-літерні маркери відсікаються.
  4. Транспонує — якщо мови в рядках, а ключі у колонках, перевертає таблицю автоматично.
  5. Маппить локалі та ключі — конвертує описові імена у ISO-коди або в зручні слаги.

Як редагувати

  • Зміни код / ключ — клік у поле input, набирай нове значення. Превʼю CSV оновлюється миттєво.
  • «пропустити» — клацай по правому посиланню. Рядок підсвітиться, не потрапить у експорт.
  • «включити» — для рядка, який адаптер сам відсіяв (зазвичай через незрозумілу назву). Впиши код вручну.

📋 Скопіювати в буфер

Натиск кнопки Скопіювати — увесь CSV у буфері. Корисно для пасту в плагін, що приймає текст.

💾 Завантажити файл

Кнопка Завантажити CSV — згенерує locality.csv з UTF-8 BOM (працює в Excel без іспанських питань про кодування).

2Інструмент: Locality плагін

Установка (одноразово)

  1. Заінсталюй UXP Developer Tool

    Безкоштовний від Adobe → developer.adobe.com

  2. Запусти UXP DevTool → Add Plugin

    Покажи на файл dist/manifest.json з папки плагіна.

  3. Натисни Load

    Photoshop підхопить плагін за секунду. У меню Plugins → Locality зʼявиться панель.

Мінімальна версія Photoshop

2024 (25.0.0) або новіша. На старших — плагін не підвантажиться, manifest заблокує запуск.

Підготовка PSD

Правило одне: імʼя текстового шару === ключ у CSV. Без пробілів зайвих.

Layers

  • 📁banner-master
  • Ttitle
  • Tsubtitle
  • Tcta
  • Tbonus
  • 🖼️bg-image

title ← у CSV ключ title

cta ← у CSV ключ cta

Шарів без перекладу торкатися не будуть — лог покаже warning, але ні шарів не зламає.

Поширена помилка: шар названо title  (з пробілом у кінці). У логу буде "title " has no translation. Перейменуй у Photoshop, прибравши хвостовий пробіл.

Завантажити переклади

⚙ Locality

URL таблиці

locality-csv...workers.dev/loc.csv

Мова

✓ Loaded 5 keys, 8 languages

Поле URL — постав сюди лінк на твій CSV. Адаптер віддає файл — заливай його куди завгодно (GitHub Pages, Cloudflare Pages, чи опублікований Google Sheets).

↻ Refresh — підтягне останню версію. Cache-busting вмонтовано — плагін завжди бачить актуальний контент.

Мова — дропдаун заповнюється з колонок CSV після Refresh.

URL та обрану мову плагін памʼятає між сесіями (зберігає в localStorage).

Apply — що насправді відбувається

  1. Обхід шарів — плагін рекурсивно знаходить усі текстові шари (включно з вкладеними у згорнуті групи).
  2. Збіг імʼя ← ключ — для кожного шару шукає рядок у CSV, де key дорівнює імені.
  3. Підстановка тексту — заміняє contents на значення обраної мови.
  4. Autofit — якщо переклад ширший за оригінал, бінарно зменшує кегль (до 60% від базового). Якщо не вистачає — попередить у логу.
  5. Шрифти — для RTL/CJK-мов автоматично підтягує Noto Sans (Arabic/Hebrew/Devanagari/CJK...) якщо вказана колонка font_family порожня.
  6. Замкнені шари — тимчасово анлокаються на час редагування, потім локаються назад.
  7. Один Undo — вся локалізація — це одна транзакція. Ctrl+Z повертає до оригіналу одним кроком.

Лог унизу панелі

Показує що зробилось і що ні. Рівні:

  • ok — шар оновлено
  • warn — шрифт не встановлено / autofit на 60% / шар без перекладу
  • skip — ключ є в CSV, але шару з таким імʼям нема в PSD
  • error — URL невалідний / нема активного документа

3Сценарії з життя

Сценарій: локалізація банера на 8 мов

Базовий кейс. Маркетинг прислав Google Sheet з 5 ключами і 8 мовами. Ти готуєш PSD-мастер, рендериш 8 версій.

CSV

key,ru,en,tr,uk,kk,ar,ja,ko
title,Майская жара,May Heat,MAYIS SICAĞI,Травнева спека,Мамыр ыстығы,حرارة مايو,メイヒート,5월 더위
subtitle,Горячий май,Red-hot May,ATEŞLİ MAYIS,Гарячий травень,Ыстық мамыр,مايو الحار,激アツの5月,뜨거운 5월
cta,Играть,Play,OYNA,Грати,Ойна,العب,プレイ,플레이
bonus,БОНУС,BONUS,BONUS,БОНУС,БОНУС,مكافأة,ボーナス,보너스
period,Май 2026,May 2026,Mayıs 2026,Травень 2026,Мамыр 2026,مايو 2026,2026年5月,2026년 5월

Що робиш

  1. PSD-мастер: 5 текстових шарів (title, subtitle, cta, bonus, period) + графіка.
  2. Прогоняєш через адаптер → завантажуєш loc.csv на Cloudflare Pages (або куди завгодно з публічним URL).
  3. У Photoshop відкриваєш мастер. У плагіні встановлюєш URL → Refresh.
  4. Дропдаун: вибираєш en → Apply → Save As banner-en.psd.
  5. Повторюєш: tr → Apply → Save As banner-tr.psd.
  6. ...×8

Сценарій: заміна команд матчу

Креатив один, контент змінюється кожен тиждень — пари команд, дати, коефіцієнти. Замість 30 файлів — 1 мастер + CSV.

CSV (variants як «локалі»)

key,match_1,match_2,match_3,match_4
team_1,Real Madrid,Manchester City,Bayern,PSG
team_2,Barcelona,Liverpool,Dortmund,Marseille
date,12.05,14.05,16.05,18.05
odds,1.85,2.10,1.65,2.35
cta,Поставити,Поставити,Поставити,Поставити

Як це працює

  1. PSD: шари team_1, team_2, date, odds, cta.
  2. Адаптер впізнає Match 1, Match 2 як варіанти. Можеш і так писати, і так — v1, variant 2...
  3. У плагіні дропдаун покаже match_1, match_2... — Apply для кожного.

Не забудь зняти випадковий пробіл у імені шару — team_1  (з пробілом) і team_1 для плагіна — це різні шари.

Сценарій: дата-варіанти однієї мови

Той самий креатив крутиться кілька днів — змінюється тільки дата. Робиш дві колонки RU і RU_2 з однаковим текстом і різними датами.

key,ru,RU_2,en,EN_2
slide_1,30/05,31/05,30/05,31/05
slide_2,Ставка без риска до $100,Ставка без риска до $100,Risk-free bet up to $100,Risk-free bet up to $100
cta,Сделать ставку,Сделать ставку,Place a bet,Place a bet

Плагін зберігає регістр для кодів з підкресленням і великою літерою — папки будуть RU_2, EN_2, не ru_2.

Сценарій: RTL + автошрифти

Креатив для CIS + GCC (арабський ринок) + азіатки. Не треба руками перемикати напрям тексту чи шукати шрифт — плагін сам.

🔄

RTL автоматично

Для ar, he, fa, ur, ckb, ps, sd, yi — направлення тексту перемикається автоматично.

🔤

Шрифт сам підставиться

Якщо мастер-шрифт не покриває скрипт, плагін підставить Noto Sans (Arabic / Hebrew / Devanagari / Bengali / Tamil / Thai / Myanmar / Khmer / Lao / Georgian / Armenian / Ethiopic / KR / JP / SC).

⚙

Або вказуєш свій

Колонка font_family у CSV перекриває автодетект. Постав туди PostScript-імʼя — для конкретної мови застосується саме воно.

4FAQ і поширені помилки

Sheet не вантажиться, помилка «HTTP 401» або «sheet приватний»

Зроби таблицю публічною: Share → «Anyone with the link» (Viewer). Якщо потрібен жорсткий контроль доступу — File → Share → Publish to web → CSV. Опублікований URL прийме адаптер.

Адаптер транспонував таблицю, а в мене вона мала залишитися як була

Це героїстика — спрацьовує коли в col 0 знайдено ≥50% мов, а в row 0 ≥50% ключів. Якщо хочеш вимкнути — налаштуй таблицю так, щоб у col 0 точно стояли ключі (title, cta, step_1...). Або правни вручну в маппінг-таблиці після парсингу.

У логу [warn] "title " has no translation

Хвостовий пробіл у імені шару. Photoshop часто додає його при копіюванні. Перейменуй шар: дабл-клік на імʼя, прибери пробіл, Enter.

У логу [skip] no layer "footer" in document

Ключ є в CSV, але шара з таким імʼям нема в PSD. Або додай шар, або видали ключ з таблиці. Це warning — інші шари оновляться нормально.

Autofit зачепив 60% — текст все одно виходить за межі

Плагін захищає тебе від нечитабельного дрібного шрифту. Варіанти: вкоротити переклад; збільшити текстову рамку у мастері; вручну зменшити кегль після Apply.

Арабський текст не відображається або «коробочки»

Photoshop не знайшов шрифт із підтримкою Arabic. Поставив Noto Sans Arabic (безкоштовний, fonts.google.com). Плагін підхопить автоматично — або вкажи його у CSV колонці font_family.

Можна локалізувати кілька шарів з однаковим імʼям?

Так — плагін зайде у ВСІ шари, чиє імʼя збігається з ключем. Корисно якщо в макеті дублі (наприклад, мобільний + десктоп варіант на одному канвасі).

А якщо я хочу різний шрифт для українського та для японського у тому самому шарі?

Колонка font_family у CSV — постав туди різні PostScript-імена для різних мов. Плагін застосує саме той шрифт при Apply. У всі інші мови — мастер-шрифт лишиться.

Чи можна автоматично експортувати всі мови як окремі файли?

Roadmap, ще не реалізовано. Зараз: Apply → Save As → повторити. У наступних версіях буде кнопка «Export all languages» — згенерує по PSD/PNG для кожної мови за одну транзакцію.

Готовий спробувати?

Адаптер — у браузері, плагін — встановлюється за 2 хвилини.
