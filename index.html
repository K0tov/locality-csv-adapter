Locality CSV Adapter

Будь-який шит → CSV для плагіна Locality

Google Sheets Excel Word PDF CSV / TSV

Як користуватись адаптером

За 4 кроки: шит → CSV → плагін Locality → банери в PSD.

  1. Підготуйте дані. Google Sheets / Excel з ключами в колонці A і мовами (або варіантами) у наступних колонках.
  2. Завантажте у адаптер. Виберіть джерело: Google Sheets URL, файл .xlsx/.docx/.pdf, або вставте CSV руками. Натисніть Розібрати.
  3. Перевірте мепинг. Адаптер автоматично визначить мови і ключі (зелені бейджі = ОК). Якщо щось жовте — впишіть код руками.
  4. Завантажте locs.csv. Або скопіюйте у буфер.
  5. У Photoshop: плагін Locality → Choose CSV… → виділіть майстер-групу → відмітьте мови → Apply.

Адаптер очікує таку структуру:

keyruentr...
titleМайська спекаMay HeatMAYIS SICAĞI...
subtitleГарячі перемогиRed-hot winsATEŞLİ KAZANÇLAR...
ctaГратиPlayOYNA...
  • Колонка A = key. Значення мають точно збігатися з іменами текстових шарів у PSD (title, subtitle, cta, bonus...).
  • Колонки B+ = мови. ISO коди (ru, en, tr) або назви мов (Русский, Англійська, English) — адаптер розпізнає.
  • Опційна колонка font_family — PostScript-ім'я шрифту що перевизначить шрифт майстра для конкретного рядка.
  • Порожні клітинки пропускаються з логом, не падає.

Окрім локалізацій, плагін підтримує варіанти кампаній — наприклад різні матчі, дати, промокоди на одному дизайні.

keyMatch 1Match 2Match 3
matchManchester vs ArsenalBarcelona vs PSGReal vs Bayern
date21.05.202628.05.202605.06.2026
promoMAY100MAY200JUNE50
titleГарячий тижденьТоп-матчФінал
  • Адаптер розпізнає патерни Match N, Variant N, V1/V2, Матч N, Варіант N.
  • Плагін генерує групи Loc/MATCH_1, Loc/MATCH_2, Loc/MATCH_3 — кожна копія майстра з заміненими `match`, `date`, `promo` тощо.
  • RTL і авто-Noto не вмикаються (це не мови) — використовується брендовий шрифт майстра.
  • Можна навіть комбінувати з мовами: колонки ru_match_1, ru_match_2, en_match_1 створять матрицю Loc/RU_MATCH_1, Loc/RU_MATCH_2, Loc/EN_MATCH_1.

Структура майстер-PSD:

Document
└── Loc                    ← обгортка (плагін створить сам)
    └── Master              ← ваша майстер-група
        ├── title            (текстовий шар)
        ├── subtitle         (текстовий шар)
        ├── cta              (текстовий шар)
        ├── match            (опційно — для варіантів)
        ├── date             (опційно)
        └── promo            (опційно)
  • Імена шарів точно як ключі в CSV: title, cta, з малої букви, без зайвих пробілів.
  • Майстер може містити будь-що: картинки, smart-objects, ефекти, маски — все скопіюється.
  • Виділіть майстер у Layers panel перед Apply, або виділіть обгортку Loc — плагін сам знайде майстер всередині.
  • Один Ctrl+Z скасовує всю генерацію (32 мовні групи зникнуть одним кроком).
Чи мої дані йдуть на сервер?
Ні, все працює у вашому браузері (JavaScript). Адаптер не надсилає жодного байта на сервер. Можна перевірити DevTools → Network.
Sheet приватний, не вантажиться
CORS блокує крос-доменні запити до приватних шитів. Опублікуйте через Файл → Поділитися → Опублікувати в інтернеті → CSV, або експортуйте у xlsx і завантажте файл.
Excel з кількома аркушами
Адаптер автоматично покаже dropdown зі списком аркушів і обере найбільший за замовчуванням. Можна вибрати інший вручну.
PDF не парситься правильно
PDF не зберігає семантику таблиць — це просто текст з координатами. Якщо результат поганий, експортуйте як xlsx з оригіналу.
Що таке UTF-8 BOM в завантаженому CSV?
Перші 3 байти EF BB BF — мітка для Excel що файл UTF-8. Без них Excel показує кирилицю як ?????. Плагін Locality читає BOM коректно.
Не розпізналась мова з мого шиту
Впишіть код руками у поле ISO КОД. Адаптер запам'ятає для цього сеансу. Або скиньте назву — додамо у словник.

1. Джерело перекладів

Якщо вставите /edit URL — інструмент спробує перетворити на публічний CSV. Якщо ваш Sheet приватний, перейдіть у Файл → Поділитися → Опублікувати в інтернеті → CSV і вставте отриманий /pub?output=csv URL.
Підтримуються: .csv, .tsv, .xlsx, .xls, .docx, .pdf. Для Word/PDF читається перша таблиця документу.
XLSX має кілька аркушів — оберіть той з перекладами.
